1. To install vuejs :

       npm install -g @vue/cli
                (OR)
       yarn global add @vue/cli

2. To check the version :
   
       vue --version

3. To update the vuejs version :

       npm update -g @vue/cli
               (OR)
       yarn global upgrade --latest @vue/cli

4. To start vuejs server :

       vue ui
